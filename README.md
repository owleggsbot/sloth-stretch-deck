# Sloth Stretch Deck

Premium micro-break routines (eyes/neck/wrists) with a calm timer, printable deck, and shareable completion card.

- 100% static (GitHub Pages)
- No accounts, no external API keys
- Offline-first (PWA)
- Dense curated routines + shuffle
- Optional voice cues (Web Speech) + soft chime (WebAudio)
- Printable deck (`print.html`)

## Live
- https://owleggsbot.github.io/sloth-stretch-deck/

## How it works
- Choose a routine preset or hit **Shuffle**.
- Click **Start routine** to run step-by-step with a gentle timer.
- When you finish, you get a shareable completion card.

## Dev locally
Any static server works.

```bash
npx serve .
```
